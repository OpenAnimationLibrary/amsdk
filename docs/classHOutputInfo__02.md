<td> </td>
<td><p>Returns the Gamma value for image data correction prior to storing to file.</p>
<p>The Gamma value returned by <a href="classHOutputInfo.md#3528c8db602d6c9459aa27410c2882c5" class="el">GetOutputGamma()</a> will match the preset. However, except for GAMMA_CUSTOM, the Gamma presets follow relatively specific ways of correcting the image data. For instance, although <a href="classHOutputInfo.md#3528c8db602d6c9459aa27410c2882c5" class="el">GetOutputGamma()</a> will return 2.2 for GAMMA_NTSC, this only an approximation. The actual correction should follow the CIE Rec 709.</p>
