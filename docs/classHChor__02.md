<td><p>Retrieves a separate HChor child list that holds all child HModels. It is important to note that this list is not sorted in the same order as it appears in the Chor folder in the PWS. When working with this list, you may not use any iteration porcess other than <a href="classHModel.md#5e9bd7932a3a4e05beeab0e21e8628ee" class="el">HModel::GetNextModelInChor()</a> to guarantee that all models are iterated through.</p>
<p>Here is a code snippet that will iterate through all HModels in an HChor object.</p>
<div class="fragment">
<pre class="fragment"><code>for (HModel *model = chor-&gt;GetChildModel(); model; model = model-&gt;GetNextModelInChor()) {
   ...
}</code></pre>
